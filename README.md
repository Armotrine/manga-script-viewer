# Manga Script Viewer

同时浏览漫画图片和对应翻译文档的桌面工具。

## 功能

- 左侧显示漫画图片，右侧显示同名的 Markdown 翻译文档
- 按文件名编号顺序浏览（如 `i-001` → `i-002` → `i-003`）
- 支持键盘、鼠标按钮导航及跳页
- 自动记住上次打开的文件夹和页码，下次启动直接恢复

## 文件命名规则

图片和 Markdown 文件使用**相同的文件名前缀**，可以放在**同一文件夹**，也可以放在**不同子文件夹**中：

```
# 同一文件夹
my-folder/
├── i-001.jpg
├── i-001.md
├── i-002.jpg
├── i-002.md
└── ...

# 子文件夹分离
my-folder/
├── image/
│   ├── i-001.jpg
│   └── i-002.jpg
└── script/
    ├── i-001.md
    └── i-002.md
```

选择 `my-folder/` 即可，应用会自动识别配对关系。

支持的图片格式：`.jpg` `.jpeg` `.png` `.webp` `.gif`

> 如果某张图片没有对应的 `.md` 文件（或反之），该页仍会正常显示，另一侧提示"无内容"。

## 使用方法

### 开发模式运行

```bash
# 安装依赖（首次运行）
npm install

# 启动应用
npm start
```

启动后，点击界面左上角的「打开文件夹」按钮，选择包含图片和翻译文件的文件夹即可开始浏览。

### 键盘快捷键

| 按键 | 操作 |
|------|------|
| `→` / `↓` / `j` | 下一页 |
| `←` / `↑` / `k` | 上一页 |
| `Home` | 跳到第一页 |
| `End` | 跳到最后一页 |

### 跳页

点击导航栏中间的页码数字，输入目标页码后按 `Enter` 跳转，按 `Esc` 取消。
